# IntelliHire UI Screenshots

Use this page as the screenshot checklist for reports, PPTs, and viva demos.

## Recommended Screenshots

1. Login page
   - Shows the IntelliHire entry point and role-based access.

2. Admin dashboard
   - Shows platform overview, user management, college/TPO/recruiter setup.

3. Recruiter dashboard
   - Shows exam creation, question management, candidate assignment, and results.

4. TPO dashboard
   - Shows student/candidate management and assigned exam tracking.

5. Candidate dashboard
   - Shows available exams, onboarding status, results, and certificates.

6. Exam attempt screen
   - Shows MCQ/coding questions, timer, and answer submission flow.

7. Results page
   - Shows score, performance summary, and certificate eligibility.

## How To Capture

1. Start the project:

```bash
npm run dev
```

2. Open:

```text
http://localhost:3000
```

3. Login with the default admin account:

```text
Email: admin@recruitexam.com
Password: admin123
```

4. Capture each screen and save the images in:

```text
docs/screenshots
```
